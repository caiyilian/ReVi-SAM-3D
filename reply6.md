# 半监督 3D 医学分割三项创新点的引言式整理稿

> 可直接用于阶段性汇报、开题答辩或后续论文引言写作。整体写法按照“问题提出—现有进展—核心不足—我们的应对”的逻辑组织，尽量保持学术化、完整可独立阅读。

在半监督 3D 医学分割中，如何从有限标注数据中学习到可迁移的解剖结构知识，并进一步稳定挖掘大量无标注体数据，始终是影响方法上限的关键问题。现有主流半监督方法大多围绕 consistency regularization、co-training 或 pseudo-labeling 展开，虽然在一定程度上缓解了标注稀缺带来的训练瓶颈，但其无标注监督通常仍来自同构小模型之间的相互约束，缺少一个能够提供更强结构先验与更高质量伪监督的教师来源。与此同时，SAM 及医学 SAM 类基础模型在少样本适应与交互式分割方面展现出明显潜力，尤其是 SAM-Med3D 这类 3D 基础模型，能够借助少量提示快速定位目标区域。然而，这类模型本质上仍是类别不可知的交互式分割器：它依赖显式提示触发目标分割，难以直接承担高效的一次前向多类别预测，也尚未自然融入半监督 teacher-student 学习框架。基于这一缺口，我们将 **SAM-Med3D 重构为训练阶段的自动交互式教师**：在无标注数据上由自动策略驱动其生成目标掩码，并通过可靠性筛选构造高质量伪监督，再将这些监督迁移到 one-shot 多类别 student，从而把基础模型的强交互能力转化为半监督场景下可部署的多类别分割能力。

然而，仅仅引入强教师并不能自动解决无标注数据的监督构造问题。对于交互式 3D 基础模型而言，提示的质量往往直接决定分割质量；而在半监督场景中，无标注样本恰恰缺乏 GT 作为提示来源，导致教师模型虽然具备潜力，却缺少稳定、可泛化的自动交互入口。已有强化学习相关工作证明，RL 可以用于边界优化、点击修正、弱监督定位或策略控制，但现有研究大多仍停留在 2D 场景、局部修补或单次交互层面，较少真正面向 **3D 体数据的自动点提示生成**，更缺少与医学 SAM 教师链路及半监督伪监督构造的系统结合。为此，我们进一步提出 **类条件顺序式自动纠错提示策略**：将自动提示建模为一个面向类别、面向当前分割状态、面向历史交互轨迹的序列决策过程，使策略网络能够在无标注体数据上自动决定“对哪个类别、在何处、以何种正负提示继续修正”。同时，我们在该策略网络中引入 **少量冻结的预训练 LLM 中间 Transformer block**，对历史提示、当前状态、类别语义与体级先验进行联合建模，并仅训练轻量适配层与策略头，以在有限医学数据条件下提升长程依赖建模能力与参数效率。这样，RL 不再只是一个附属优化器，而成为驱动自动教师稳定工作的关键决策核心。

进一步地，3D 医学影像的训练通常基于 patch 或局部子体进行，这虽然提高了计算可行性，却也使模型容易受限于局部感受野，难以获得体数据层面的整体语义约束。现有视觉-语言或大模型先验方法已经证明，高层语义信息能够帮助缓解类别混淆、缩小搜索空间，并为分割提供更强的任务相关引导；但这类方法多数仍集中在 2D 图像、单切片文本注入或任务级提示层面。对于 3D 医学体数据而言，如何在 2D 医学 VLM 的输入限制下，构造真正能够服务于整个 volume 的稳定语义先验，仍缺少充分研究。尤其是在自动提示场景中，若缺乏体级语义支持，策略网络往往只能依据局部误差进行短视探索，容易在无关切片和不存在类别上产生低效搜索。针对这一问题，我们提出 **基于代表切片选择与语义融合的 volume-level semantic prior 构造机制**：通过对 3D volume 中具有代表性的切片进行选择、提取片级语义，并进一步融合形成体级先验表示，用于向策略网络提供类别存在倾向、搜索优先级以及更具解剖上下文的全局语义约束。该设计并不试图让 2D VLM 直接“理解”完整 3D 体数据，而是以更稳妥的方式在 2D 语义能力与 3D 决策需求之间建立桥梁。

为了解决上述问题，我们提出了一个面向 **半监督 3D 医学分割** 的统一框架：首先，利用 **代表切片选择与语义融合** 构造 volume-level semantic prior，为 3D 体数据提供可服务于自动提示的体级语义支撑；其次，在此基础上设计 **类条件顺序式自动纠错提示策略**，并借助冻结 LLM 中间层对历史提示、当前状态、类别语义和体级先验进行联合建模，从而在无标注数据上自动生成高质量 3D 点提示；最后，以该策略驱动 **SAM-Med3D 作为自动交互式教师**，在无标注样本上构造经过可靠性筛选的伪监督，并将其迁移到 one-shot 多类别 student 中，实现“体级先验引导—顺序策略决策—自动教师生成伪监督—student 高效部署”的完整闭环。该框架的核心价值不在于简单叠加多个强模块，而在于把基础模型、大模型语义先验与强化学习决策有机组织起来，使无标注数据真正转化为稳定、可利用的训练信号。
