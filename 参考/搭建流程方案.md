# ReVi-SAM-3D 训练框架搭建详细拆解流程

为了保证代码的高质量和高可控性，我们将整个工程拆解为以下循序渐进的微小步骤。每个步骤都遵循“高内聚、低耦合”的原则，确保每完成一步都可以独立验证。
为了避免反复查阅多个文档，这里将每一步**可以复用哪些项目的代码、具体如何使用**进行了详细标注。

**步骤 6：集成 CLIP/BERT 文本特征提取**

## 阶段 2：魔改 SAM 2 (创新点 1 & 2)

**步骤 7：实现跨维度注意力投影模块 (创新点 1 核心)**
- **动作**：在 `models/sam2_modified/projection.py` 中，编写 `DeformableProjectionModule` 类。
- **实现细节**：完全由我(AI)编写。内部包含 Cross-Attention (Q:视觉, K/V:文本)，然后将其输出喂给您已在服务器环境配置好的 **`DCNv4`** 算子（`from DCNv4 import DCNv4`），实现 1D 文本向 2D 视觉特征的注入。
- **当前状态**：✅ **已完成** (代码位于 `ReVi-SAM-3D/models/sam2_modified/projection.py`)

**步骤 8：将投影模块注入 SAM 2 Image Encoder**
- **动作**：修改拷贝过来的 SAM 2 `image_encoder` 源码。在特定的 ViT Block 之后，插入步骤 7 编写的 `DeformableProjectionModule`。
- **验证**：输入一张图像和一段 Mock 文本，跑一次前向传播，确保不报错且维度对齐。
- **当前状态**：✅ **已完成** (代码位于 `ReVi-SAM-3D/models/sam2_modified/sam2-main/sam2/modeling/sam2_base.py` 和 `test_step8_sam2_injection.py`)

**步骤 9：提取并冻结 LLaMA 单层 (创新点 2 & 3 基础)**
- **动作**：在 `models/common/llm_extractor.py` 中，完全借鉴《LLM4Seg》的“三明治”结构（Adapter1 -> Frozen LLM Layer -> Adapter2），剥离出轻量级 LLaMA 的单层用于后续特征提取。
- **复用说明**：
  - **来源项目**：参考 `大模型分割代码\LLM4Seg-main\network\llm4seg.py` 中是如何提取预训练 LLaMA 层的。
  - **怎么用**：使用 `transformers` 库 `AutoModelForCausalLM.from_pretrained(...)`，取出 `model.model.layers[layer_idx]`，并将其所有参数的 `requires_grad` 设为 `False`。编写适配输入维度的 `Linear` 投影层（Adapter），并生成配套的 2D RoPE 位置编码。
- **当前状态**：✅ **已完成** (代码位于 `ReVi-SAM-3D/models/common/llm_extractor.py` 和 `test_step9_llm_extractor.py`)

**步骤 10：重写 SAM 2 Memory Attention 模块 (创新点 2 核心)**
- **动作**：修改 `ReVi-SAM-3D/models/sam2_modified/sam2-main/sam2/modeling/memory_attention.py` 中的 `MemoryAttention` 类。
- **实现细节**：由我(AI)编写。在初始化和 `forward` 中动态注入拦截逻辑：
  1. 将“前一切片的记忆特征(Memory)”和“当前切片的特征(Current)”在 Token 维度拼接 (`[B, M+N, C]`)，送入步骤 9 的 LLM 层（全局分支），利用 LLM 强大的长程序列匹配能力找到时空对应关系，并切片截取当前帧增强后的特征。
  2. 旁路并联调用 `DCNv4`（局部分支），对当前帧特征进行微观几何形变对齐。
  3. 将全局与局部特征融合并投影回原维度，**直接跳过/替换** SAM 2 原本的多层纯视觉 Transformer。
- **当前状态**：✅ **已完成** (代码已注入 `memory_attention.py`，测试脚本为 `test_step10_memory_attention.py`)

**步骤 11：替换 SAM 2 的原生追踪器并实现双向 Wrapper**
- **动作**：在 `ReVi-SAM-3D/models/sam2_modified/sam2-main/sam2/sam2_video_predictor.py` 的 `SAM2VideoPredictor` 类中，注入 `bidirectional_propagation` 函数。
- **实现细节**：由我(AI)编写。该函数接收整个 3D 切片序列和带有 Bbox 的初始切片索引。它会调用底层的 `propagate_in_video` 两次：一次 `reverse=False` 向 Z 轴正方向传播；一次 `reverse=True` 向 Z 轴负方向传播。最后将所有 2D Mask 收集并拼装成完整的 3D预测结果。
- **当前状态**：✅ **已完成** (代码已注入 `sam2_video_predictor.py`，测试脚本为 `test_step11_bidirectional_wrapper.py`)

## 阶段 3：构建强化学习闭环 (创新点 3)

**步骤 12：构建大模型赋能的轻量级 RL 策略网络**
- **动作**：在 `ReVi-SAM-3D/models/rl_agent/llm_policy.py` 中编写 `LLMDrivenPromptAgentDDQN`。
- **实现细节**：由我(AI)编写。该网络接收两个状态：SAM 2 提取的 2D 视觉特征图和当前边界框 (Bbox)。它将两者转化为 Token 并拼接，送入复用的冻结 LLM 层（DeepSeek）。利用 LLM 的全局注意力提取融合了视觉上下文的 Bbox 特征，最后通过一个轻量级 MLP 输出 9 个离散动作的 Q 值。
- **当前状态**：✅ **已完成** (代码位于 `llm_policy.py`，测试脚本为 `test_step12_llm_policy.py`)

**步骤 13：移植并适配 DQN 核心算法**
- **动作**：在 `ReVi-SAM-3D/models/rl_agent/dqn_agent.py` 中，移植 DQN 骨架。
- **复用说明**：
    - **来源项目**：`强化学习分割论文代码\DRL-Enhanced-Interactive-Segmentation-PyTorch-Implementation--main` (即 RCI-Seg)。
    - **实现细节**：复用其 `ReplayBuffer` (经验回放池) 和 $\epsilon$-Greedy 探索逻辑。针对我们的任务进行了魔改：(1) 将原本的单图片 State 修改为 `(image_features, current_bbox)` 的双元组状态；(2) 用步骤 12 的 `LLMDrivenPromptAgentDDQN` 替换了普通的 CNN 策略网络；(3) 将原本激进的 Target Network 硬更新修改为更稳定的**软更新 (Soft Update, 引入 $\tau$)**。
- **当前状态**：✅ **已完成** (代码位于 `dqn_agent.py`，测试脚本为 `test_step13_dqn_agent.py`)

**步骤 14：编写 Dice 评估与辅助指标函数**
- **动作**：在 `utils/metrics.py` 中，编写计算 Dice Similarity Coefficient (DSC) 的函数，这将是后续 RL 奖励计算的基石。可以复用 `DDSP-SSECP` 中的 Dice 计算代码。
- **当前状态**：✅ **已完成** (代码位于 `utils/metrics.py`，测试脚本为 `test_step14_metrics.py`)

**步骤 15：搭建 RL 交互闭环环境 (ClosedLoopEnvironment)**
- **动作**：在 `env/sam_closed_loop.py` 中，封装强化学习环境。
- **复用说明**：
  - **来源项目**：参考 `强化学习分割论文代码\SPARS-main` 中的 `CursorImageEnv` 环境设计模式。
  - **怎么用**：借鉴它继承 `gym.Env` 的外壳，但内部的 `step` 函数由我(AI)完全重写：接收动作(调整 Bbox) -> 调用魔改 SAM 2 预测 -> 计算 DSC 变化 -> 返回作为 Reward。
- **当前状态**：✅ **已完成** (代码位于 `env/sam_closed_loop.py`，测试脚本为 `test_step15_env.py`)
