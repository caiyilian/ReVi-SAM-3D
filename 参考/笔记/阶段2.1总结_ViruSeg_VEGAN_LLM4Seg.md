## 1. 当前领域的痛点 (Pain Points)

### 1.1 数据匮乏与模型训练的矛盾
- **极端的数据稀缺 (Severe data scarcity)**：医学图像（尤其是TEM病毒图像、显微图像等）需要专家进行高成本和劳动密集型的注释。标注数据的匮乏严重限制了传统监督学习模型的性能，也很难从头训练出具有强大泛化能力的 Transformer 模型。
- **过拟合风险高**：在数据匮乏的场景下，复杂的混合模型（如CNN-Transformer）的 Transformer 块如果从头训练，很容易过拟合，无法学习到鲁棒的语义表示。

### 1.2 医学图像固有的复杂性
- **形态差异大 (Morphological variability)**：目标对象（如病毒、细胞）的形状、大小和形态变化极大，甚至同类目标之间也存在很大差异。
- **边界模糊与相似性干扰**：医学图像通常对比度低、噪声高、目标密集（如重叠的细胞），且目标边界模糊。同时，不同类别的目标可能具有极高的形态相似性，导致模型容易出现高误检率和漏检率。
- **类别不平衡**：前景（病灶/细胞核）与背景之间存在严重的类别不平衡。

### 1.3 现有模型的局限性与“性能-效率”困境
- **CNN 缺乏全局感知**：主流的 CNN 模型（如 U-Net）擅长局部特征提取，但缺乏鲁棒的全局语义建模能力，且往往难以保留目标的细粒度边界细节。
- **VLM 依赖多模态对齐**：现有的视觉语言模型（VLM）通常将 LLM 用作共享解码器，依赖复杂的跨模态对齐和“图像-文本”成对数据，难以直接应用于纯视觉任务。
- **计算成本高昂**：为了追求更高的分割精度，SOTA 模型往往伴随着可训练参数和计算开销的急剧增加，这限制了它们在资源受限的临床场景中的实际部署。轻量级模型则往往会牺牲结构和边界分割的精度。

## 2. 有价值的闪光点 (Valuable Innovations/Ideas)

### 2.1 大模型作为“特征提取器/语义增强器”的降维打击
- **无需文本标签的跨模态语义迁移**：《Pre-Trained LLM is a Semantic-Aware and Generalizable Segmentation Booster》证明了，即使是**仅在文本数据上训练的冻结 LLM**，也能直接处理视觉 token，利用其强大的全局语义理解能力来增强医学图像分割，而**不需要任何文本提示或图像-文本配对数据**。这对于我们“只有掩码标签”的约束条件是非常完美的契合。
- **极简参数微调 (Minimal-parameter frozen LLM)**：通过将 CNN 提取的局部特征映射到 LLM 的维度，经过冻结的 LLM 层进行全局语义建模，再映射回视觉维度，这种“即插即用”的设计不仅带来了极低的参数增加，还避免了从头训练的过拟合风险。
- **利用 CLIP 的跨模态语义增强 U-Net**：《VEGAN》将预训练的 CLIP（无需微调）生成的跨模态语义特征，直接融合到 Attention U-Net 的跳跃连接中，从而在无需额外监督的情况下，增强了模型的语义理解和跨数据集泛化能力。

### 2.2 针对形态与边界的特定架构设计
- **多尺度与注意力机制结合**：在特征提取阶段，结合 FPN（特征金字塔网络）或多尺度注意力机制，可以有效应对目标大小不一的问题。
- **边缘增强双输入**：《VEGAN》使用传统算子（Sobel, Canny, Laplacian）融合生成的边缘图作为模型的额外输入，以增强模型对模糊边界和重叠目标的感知。这是一种无需额外标注的廉价“先验”。
- **特定网络层优化**：《ViruSeg》针对大模型主干网络进行了优化，例如使用 Sub-Layer Normalization (sub-LN) 提升细节识别精度，使用 2D Rotational Position Embedding (RoPE) 提供更准确的位置参考。

### 2.3 训练范式的创新
- **级联分割框架**：《ViruSeg》将大模型提取的多尺度特征输入到级联 Mask R-CNN (CMR) 中，通过多阶段级联回归逐步细化边界框，输出高精度的像素级掩码。
- **对抗生成网络 (GAN) 辅助分割**：《VEGAN》采用 Pix2Pix cGAN 框架，使用 PatchGAN 作为判别器在 patch 级别评估掩码的真实性，促使生成器（分割网络）产生更精细的结构细节和更清晰的目标边界，这对于解决医学图像边界模糊问题提供了一个有趣的思路。
