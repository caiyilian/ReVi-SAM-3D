## 1. 当前领域的痛点 (Pain Points)

### 1.1 大模型在密集预测任务中的“未被充分利用”
- **过度依赖文本标签/提示**：先前的 VLM 工作大多依赖文本引导、显式类标签语义监督或 Prompt 工程，需要额外的语言标注，不是纯粹的数据驱动。
- **计算成本高昂**：大模型的全面微调（Full fine-tuning）对于资源受限的临床用例来说成本过高，而如果不微调，大模型通常只关注图像级分类，难以直接应用于体素级/像素级的密集预测（Dense Prediction）。
- **盲目扩大参数量**：现有的模型改进逻辑往往将性能提升归因于模型复杂度和参数量的增加，而不是有效的特征细化，这种扩展策略在数据匮乏的临床环境中是不切实际的。

### 1.2 弱监督/半监督学习的固有缺陷
- **单一弱监督信号的局限性**：例如临床医生注视点（Gaze），虽然能指示“看哪里”，但存在固有的稀疏性、噪声（分心），且缺乏清晰的病变边界定义。而 VLM 虽然能解释“为什么看”，但存在领域差距和语义幻觉，缺乏细粒度分割所需的像素级精度。
- **空间定位与语义可解释性之间的鸿沟**：现有的弱监督方法无法有效融合视觉注意力（空间）和 VLM 语义解释（语义），难以兼顾标注效率、分割精度和临床可解释性。
- **Dropout 扰动的不可控性**：在半监督学习中，基于特征级 Dropout 的一致性学习是主流范式，但其对 Dropout Rate 极度敏感。高 DR 导致 KL 散度无界增长和模型崩溃，低 DR 则正则化效果微乎其微。
- **向量量化（VQ）的信息丢失**：虽然离散特征空间可以实现更结构化的扰动，但 VQ 离散化过程固有地会导致细粒度视觉细节和高级语义信息的丢失。

## 2. 有价值的闪光点 (Valuable Innovations/Ideas)

### 2.1 创新的大模型集成范式（无需文本，参数高效）
- **LoRA 降维映射与冻结 LLM 结合**：《MedVisionLlama》提出了一种完全无需 Prompt、无需语言监督的 LLM 集成范式。它将冻结的预训练 LLM Transformer 块作为 ViT 编码器和解码器之间的“残差注意力增强器”。通过引入两个可训练的 LoRA 维度映射层来解决特征维度不匹配问题，同时选择性地在 LLM 块内应用 LoRA 进行轻量级自适应。这证明了 LLM 的预训练关系先验可以直接用于视觉密集预测，且具有极高的参数效率。
- **结构化 Prompt 约束大模型输出**：《From Gaze to Insight》在离线预处理阶段，通过临床约束的结构化 Prompt 强制 VLM 输出标准化的 JSON 对象（位置、边界、特征等），有效缩小了自然图像到医学图像的领域差距，减轻了语义幻觉。这为我们“离线使用大模型”提供了一个极佳的落地思路。

### 2.2 跨模态语义对齐与知识蒸馏
- **教师-学生架构解耦多模态监督**：《From Gaze to Insight》设计了教师-学生框架。教师模型利用高可靠性注视点和 VLM 文本语义（通过多头交叉注意力注入视觉特征）学习跨模态表示；学生模型则在更广泛、更嘈杂的数据上训练，通过特征一致性、置信度加权一致性等策略，吸收教师的知识。
- **基础模型引导的 Patch 级对比学习**：《VQ-Seg》中的 Post-VQ Feature Adapter (PFA) 模块，利用冻结的视觉基础模型（如 DINOv2）提取语义嵌入，通过 Patch 级的对比学习，将量化后的离散特征与基础模型的语义特征进行对齐，完美补偿了 VQ 过程中的语义信息丢失。

### 2.3 特征扰动与自监督的创新
- **量化扰动模块 (QPM)**：《VQ-Seg》放弃了传统的 Dropout，在离散的 VQ 码本空间中，基于码字之间的距离定义条件转移概率来打乱码本索引。这种结构化的扰动在数学上是有界且稳定的，解决了 Dropout 的不可控问题。
- **双分支共享空间与图像重建自监督**：《VQ-Seg》在量化后的特征空间中并行进行图像重建和语义分割，利用重建任务作为自监督信号，促使编码器保留完整的结构信息。
